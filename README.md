# Research Paper Summarizer

An AI-powered application that summarizes research papers and generates keywords, flashcards, and important questions.

## Features
- Upload PDF research papers
- AI-generated summaries
- Keyword extraction
- Important question generation
- Flashcards
- Download summaries

## Technologies
Python, Streamlit, Gemini API, NLP, PyPDF

## Run
pip install -r requirements.txt
streamlit run app.py
